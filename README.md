# Pendaftaran-siswa-baru-dengan-PHP
Form pendaftaran siswa baru saya buat dengan menggunakan PHP.
Saya letakkan file index.php ke dalam folder htdocs pada XAMPP folder.
Buka browser dan paste http://localhost/pendaftaran/index.php.
Saya run kode tersebut menggunakan XAMPP di local.
Jadi, mohon copas code terlebih dahulu dan run menggunakan Apache pada XAMPP di local
Setelah running akan muncul form pendaftaran beserta daftra siswa yang bisa diedit dan dihapus.
Data akan otomatis disimpan pada csv.
